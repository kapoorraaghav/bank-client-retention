Web Application

The web application was built using Streamlit so that users can access and interact with the customer churn prediction models effortlessly. The web interface lets users input information like age, credit score, account balance, number of products used and the status of account. Upon entering the information the web application runs the four models that were trained: ANN, TabNet, FT-Transformer and Autoencoder+Classifier and the prediction of each model is displayed along with the ensemble prediction. Visualizations that aid in understanding the predictions like the category of risk, model comparison and visualizations are all accessible from the web interface. This web application was constructed with Python with the aid of Streamlit, TensorFlow, NumPy, Pandas and Matplotlib
